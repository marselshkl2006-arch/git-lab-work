Проект химическая лаборатория QT  C++ v 2.1 

Проект помогающий в складском учёте химикатов с интеллектуальной задумкой правильного распределения за счёт следования определенным правилам 

Этот проект  написан только под linux системы в дальнейшем возможно расширение на windows 

МИНИМАЛЬНЫЕ СИСТЕМНЫЕ ТРЕБОВАНИЯ 

двухъядерный процессор 4-8 ГБ оперативной памяти размер внутренней памяти 1 Гб 

УСТАНОВКА КОМПИЛЯЦИЯ И ЗАПУСК
Нужно скачать папку с репозитория распаковать её в директиве пользователя 
/home/Unix-User/Chemical_lab_system/ChemicalLabQt$ 

Затем ввести команды 
mkdir build 
cd build 
cmake ..
make -4j
./ChemicalLabQt

Если же по какой-то причине компиляция не удалась по пути
 home/unix user/Chemical_lab_system/ChemicalLabQt/build 
Лежит скомпилированный бинарнник нужно открыть в терминале

ПО готово к работе добро пожаловать
